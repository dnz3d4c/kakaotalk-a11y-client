# AI 도구 활용 보고서

## 개발 방식

이 프로젝트는 **바이브 코딩** 방식으로 개발되었습니다:

| 역할 | 담당 | 비고 |
|------|------|------|
| 문제 정의 | 인간 | 시각장애인 접근성 요구사항 |
| 기술 방향 | 인간 | 이미지 인식 vs UIA 등 결정 |
| **코드 작성** | **AI** | Claude Code 활용 |
| 실사용 검증 | 인간 | NVDA 스크린 리더로 동작 확인 |
| 문서 작성 | 협업 | 인간 + AI |

### 사용 도구

- **Claude Code** (Anthropic): 코드 생성, 리팩토링, 디버깅

## 라이선스

MIT 라이선스 선택 - 접근성 도구의 최대 확산을 위해 제약 최소화.

## 정리

인간이 문제 정의와 기술 방향을 결정하고, AI가 코드를 작성하며, 인간이 실사용으로 검증하는 방식으로 개발되었습니다.
